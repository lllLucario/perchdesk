# Foundations

## Brand Direction

PerchDesk uses a green editorial system derived from the design principles documented in [DESIGN.md](/Users/kkkadoya/Desktop/perchdesk/docs/design/DESIGN.md).

The palette should communicate:
- calm focus
- spatial trust
- natural order
- quiet professionalism

It should not communicate:
- neon tech optimism
- fintech aggression
- generic productivity dashboard energy

## Typography

- Headings use serif and should feel composed rather than dramatic
- UI text uses sans-serif and should stay efficient and readable
- Monospace is reserved for technical content only
- Body copy should keep generous line height, especially in descriptive booking or discovery surfaces

## Spacing

- Base unit: 8px
- Common spacing rhythm: `8 / 12 / 16 / 24 / 32 / 40 / 56 / 80`
- Major layout rhythm matters more than local micro-adjustments

## Radius & Elevation

- Common controls: 8px
- Inputs and primary CTA: 12px
- Featured containers: 16-24px
- Prefer ring or border depth over heavy shadows

## Theme Rule

If a token or styling choice is uncertain, keep these priorities:
1. clarity of booking task
2. tonal consistency with the green editorial palette
3. restrained emphasis
4. softness over sharpness
